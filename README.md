This is a repository of data used in the manuscript "Urbanization increases gross primary production and biomass of Atlantic Forest fragments" by Wanda da Silva, Marcelo Alves, and Rafael Zenni.

The file "adapted_engine_script.js" contains a JavaScript code adapted to obtain data from the MODIS sensor products by "Google Earth Engine" - GEE (https://earthengine.google.com/).

The folder "Modis_raw_chart_data" contains csv tables with raw data of GPP and biomass obtained from charts generated in Google Earth Engine using MODIS sensor products.

the folder "QGIS_map" contains shapefiles of the studied forest fragments and TIF images with average values (between 2004 ans 2024) of GPP and biomass generated in GEE using MODIS sensor products.

The file "production_data_2004-2024.csv" contains a csv table with data extracted from the charts generated in GEE and adapted for daily values ​​of gross and net primary production and biomass gain.

The file "CO2.csv" presents annual values, in tons, ​​of total CO2 emissions in the surroundings of the studied urban and non-urban forests adapted from "Sistema de Estimativas de Emissões e Remoções de Gases de Efeito Estufa" (SEEG).
